Each column in the grid is defined using a column definition. A list of column definitions is passed into the grid for rendering.

Attribute Description
displayName The name to render in the column header
field The field of the row to get the cells data from
width Initial width, in pixels, of the cell
cellCss An object of css values.
cellCssFunc A function that when called, returns an object of css values.
cellRenderer A function for rendering a cell.
filterCellRenderer A function for rendering a cell in the filter menu.
filterCellHeight How high each cell in the filter menu, in pixels.

cellCss
Use this if a predetermined style is to be applied to all cells in the column. Eg 
var colDef = {
    name: 'Col Name',
    field' 'Col Field',
    cellCss: {color: red, background-color: green}
}
cellCssFunc
A function that when called, returns an object of css values. 
var colDef = {
    name: 'Col Name',
    field' 'Col Field',
    cellCssFunc: function(value, data, colDef, $childScope) {
                 if (value=='Police') {
                     //mark police cells as red
                     return {color: red, background-color: green};
                 } else {
                     return null;
                 }
             }
}
value: The value to be rendered.
data: The row (from the rowData array, where value was taken) been rendered.
colDef: The colDef been rendered.
$childScope: If compiling to Angular, is the row's child scope, otherwise null.
cellRenderer
A function that when called, returns HTML (either a Dom or Node element, or an HTML string) 
var colDef = {
    name: 'Col Name',
    field' 'Col Field',
    cellRenderer: function(value, data, colDef, $childScope) {
                    if (value=='Police') {
                        return 'POLICE';
                    } else {
                        return value.toUpperCase();
                    }
                }
}
value: The value to be rendered.
data: The row (from the rowData array, where value was taken) been rendered.
colDef: The colDef been rendered.
$childScope: If compiling to Angular, is the row's child scope, otherwise null.
