Why Angular Grid

Angular Grid is another Grid for HTML 5 Angular development. I created Angular Grid out of frustration because I couldn't find a grid that was very fast (sorry ui-Grid), allowed pinned columns (sorry Slick Grid) had excel like advanced filtering and was native to AngularJS (sorry pretty much every grid).

So I thought, how hard can it be? I guess I am finding out ...

Features

  • Native to Angular JS
  • Excel Like Advanced Filtering
  • Gmail like 'Quick Filter'
  • Pinnable Columns
  • Row Virtualisation
  • Column Resizing
  • Sorting
  • Grouping
  • Custom Cell Renderers
  • Auto Resizable
  • Works with 100% width & height
  • Customisable via CSS
  • No Dependencies
  • Supports very large data sets
  • Very fast

Who Should Use

I work for a major investment bank. We required a grid that can handle very large data sets and give a professional Excel-like experience to our users. Angular Grid has been developed for this purpose.

If your requirements are like ours, then you should find Angular Grid very useful. If you find another grid that is better suited to your needs, then use the other grid.
