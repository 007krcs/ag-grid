Angular Grid (ag-Grid) is another Grid for HTML 5 Angular development. I created Angular Grid out of frustration because I couldn't find a grid that was very fast (sorry ui-Grid), allowed pinned columns (sorry Slick Grid) had excel like advanced filtering and was native to AngularJS (sorry pretty much every grid).
So I thought, how hard can it be? So I gave it a shot, the latest release has the following:
  • Native to Angular JS
  • Excel Like Advanced Filtering
  • Gmail like 'Quick Filter'
  • Pinnable Columns
  • Row Virtualisation
  • Column Resizing
  • Sorting
  • Custom Cell Renderers
  • Auto Fit Size (ie width & height to 100% works, and changing width and height)
  • Very Customisable via CSS Look and Feel
  • No Dependencies (apart from core Angular), no JQuery, no jqLite
  • Supports very large data sets
  • Very very fast (approx 0..60 mph in 0.024 seconds)
Grouping will be implemented at some point, the design has it ready to slot in, just don't need it right now for my current project, so I am not in a rush.
